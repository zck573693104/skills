# Weather Skill

通过城市名称获取实时天气信息（使用高德地图API）。

## 功能特性

- 支持中文城市名输入（如“北京”、“广州”）
- 需要配置高德地图API Key
- 输出详细的中文天气信息，包含天气状况、温度、湿度、风速、风向和更新时间
- 默认位置为 **广州**，用户不输入参数时自动查询

## 使用方式

```

bash
python scripts/main.py [city]
```
### 参数说明

| 参数 | 类型 | 是否必填 | 说明 |
|------|------|----------|------|
| `city` | 字符串 | 否 | 城市名称（如 `北京`）。未提供时默认查询广州天气。 |

### 环境配置

需要设置高德地图API Key：
```
bash
export GAO_DE_API_KEY=your_api_key_here
```
### 示例

```

# 查询默认城市（广州）
python scripts/main.py

# 查询北京天气
python scripts/main.py 北京

# 查询上海天气
python scripts/main.py 上海
```

- 需要申请并配置高德地图Web服务API的Key
- 使用 `argparse` 处理命令行参数
