# 物流业务FAQ知识库

## 一、报价相关

**Q: 零担和整车怎么选择？**  
A: 一般货重超过5-6吨，或货物体积超过9.6米车容积的60%以上，选整车更划算。5吨以下选零担。对于时效敏感的货物，整车比零担快1-2天。

**Q: 报价单有效期多久？**  
A: 正常报价有效期7天。受油价和运力波动影响，旺季（1月、6-7月、11月）报价有效期缩短为3天。客户签约后按合同价执行，不受市场波动影响。

**Q: 大件/超限货物怎么报价？**  
A: 单件超过3米长度，或重量超过1吨的货物，需单独报价。请提供货物尺寸（长×宽×高）、重量、起止地点，我们专项团队24小时内回复。

**Q: 为什么收到的运费和报价单不一样？**  
A: 可能原因：①实际货重/体积超过报价时提供的数据；②加收了燃油附加费；③目的地为偏远地区产生附加费。发票会附明细说明，有疑问可联系客服核对。

## 二、时效相关

**Q: 报的时效是保证到达的吗？**  
A: 正常情况下时效承诺兑现率98%以上。以下情况时效顺延：恶劣天气（暴雪/台风）、节假日、突发道路管控。如因我方原因延误，按合同赔偿条款处理。

**Q: 可以做加急/当日达吗？**  
A: 上海、北京、广州、深圳之间的重点线路可提供加急服务，费用在标准价基础上上浮30%-50%。需提前2小时预约，不保证每天有资源。

**Q: 春节期间还发货吗？**  
A: 春节期间（除夕至初七）大部分线路正常运营，时效延长1-2天。初八全面恢复正常。具体可关注我们的公众号节假日公告。

## 三、禁运/限运品

**Q: 哪些货物不能运？**  
A: **绝对禁运**（不接）：
- 易燃易爆品（烟花、炸药、化工危品）
- 武器弹药
- 走私违禁品
- 活体动物（特殊许可除外）
- 尸体、骨灰

**Q: 哪些货物需要特殊申报？**  
A: **限运品**（需提前告知，可能加收附加费）：
- 锂电池（需UN认证及申报表）
- 液体类（需防漏包装，单票不超50升）
- 超大超重件（单件>1吨或长度>4米）
- 艺术品/古董（需保价，最高赔偿货值80%）
- 精密仪器（需填写特殊货物申报表）

**Q: 食品可以运吗？**  
A: 普通包装食品可运，但不保温，不建议运送需要冷链的生鲜食品。需要冷链服务请咨询冷链专线。

## 四、保价与理赔

**Q: 不购买保价，货损了怎么赔？**  
A: 按运费的3倍赔偿，不超过货值。建议对货值超过5,000元的货物购买保价。

**Q: 购买保价后货损怎么赔？**  
A: 购买保价后，按实际损失（凭发票）的80%赔偿，最高不超过申报价值。

**Q: 理赔流程是什么？**  
A: 
1. 签收时当场验货，如有破损拍照存档，当场拒签或注明破损
2. 48小时内向客服提交理赔申请（运单号+照片+货物价值证明）
3. 客服3个工作日内核实
4. 核实通过后5个工作日内到账

**Q: 签收后发现货损还能理赔吗？**  
A: 签收后72小时内发现内部货损可申请理赔，需提供开箱视频为证。超过72小时不予受理。

## 五、操作流程

**Q: 如何下单？**  
A: ①手机APP下单；②微信小程序下单；③电话通知专属客服代下单；④对接TMS系统API自动下单（大客户）

**Q: 几点可以上门取货？**  
A: 工作日8:00-20:00可安排取货，需提前2小时预约。20:00后的取货按夜间取货费计算（加收100元）。

**Q: 运单可以修改吗？**  
A: 货物未装车前可免费修改。已装车后修改地址费用100元，已发出后无法修改，只能退货（需承担退回费用）。

## 六、对接与系统

**Q: 支持系统对接吗？**  
A: 支持API对接，提供RESTful接口，包括：下单、查单、运单追踪、费用查询、电子发票申请。文档请联系商务获取。

**Q: 有电子回单吗？**  
A: 支持电子签收回单，签收后30分钟内可在系统查看，可导出PDF存档。
