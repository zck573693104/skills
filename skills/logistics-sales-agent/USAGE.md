# 物流销售 Agent 使用指南

## 📋 功能概览

本 Skill 为物流公司内部销售团队提供全方位的销售辅助，包括：

1. **智能运价查询** - 支持零担(LTL)和整车(FTL)报价
2. **实时运单追踪** - 查询货物当前位置和状态
3. **客户管理** - 查看客户信息、跟进提醒
4. **销售话术** - 应对客户异议的专业话术
5. **业务FAQ** - 快速解答常见问题
6. **报价单生成** - 自动生成正式报价文档

## 💬 销售话术

### 场景 1: 价格异议处理

**用户输入**：
```
客户说我们价格贵怎么办？
```

**Agent 回复**：
```
💡 价格异议处理话术：

【话术模板】
"王总，我理解您对价格的关注。不过我想跟您分享几点：

1️⃣ **价值对比**：我们的准时送达率是98%，比行业平均高5个百分点。对于电商来说，准时意味着更好的用户体验和更少的售后成本。

2️⃣ **隐性成本**：便宜5%但延误1天，可能导致您的客户投诉、退货，这些隐性成本远高于运费差价。

3️⃣ **增值服务**：我们提供24小时客服、实时轨迹追踪、快速理赔通道，这些都是包含在运费里的。

4️⃣ **长期合作**：如果您月运费能达到10万，我们可以申请9折优惠；达到20万，可以谈到8.5折。

要不我先给您做个详细的成本对比分析？看看实际能省多少、风险在哪。"

【使用要点】
- 不要直接降价，先强调价值
- 用数据说话（准时率、破损率）
- 引导客户关注总成本，而非单一运费
- 给出阶梯折扣方案，激励增量
```

## ❓ 业务FAQ

### 场景 1: 禁运品查询

**用户输入**：
```
锂电池能运吗？
```

**Agent 回复**：
```
🔋 锂电池运输规定：

✅ **可以承运**，但需满足以下条件：

1️⃣ **包装要求**：
   - 必须使用UN认证的锂电池专用包装
   - 单个电池功率 ≤ 100Wh
   - 外包装需贴"锂电池"警示标签

2️⃣ **文件要求**：
   - 提供MSDS（材料安全数据表）
   - 提供UN38.3测试报告
   - 填写危险品申报单

3️⃣ **运输方式**：
   - 仅限陆运，不可空运
   - 需走危险品专线
   - 时效比普通货物慢1-2天

4️⃣ **费用**：
   - 基础运费 + 20%危险品附加费
   - 最低收费：200元/票

❌ **不可承运的情况**：
   - 损坏/鼓包的锂电池
   - 无认证文件的锂电池
   - 功率 > 100Wh的大型电池组

💡 建议：首次合作建议先小批量试单，确保包装和文件合规。
```

```
货损了怎么理赔？
```

**Agent 回复**：
```
🛡️ 理赔规则说明：

【未保价货物】
- 赔偿标准：运费的3倍
- 举例：运费1000元 → 最高赔偿3000元
- 适用：货值较低、风险可控的货物

【已保价货物】
- 赔偿标准：货值的80%
- 保价费：货值 × 0.3%，最低50元
- 举例：货值10万 → 保价费300元 → 赔偿8万
- 适用：高价值货物、精密仪器

【理赔流程】
1️⃣ **发现货损**：签收时当场验货，拍照取证
2️⃣ **提交申请**：72小时内提交理赔申请
3️⃣ **提供材料**：
   - 运单原件
   - 货损照片（外包装+内物）
   - 货物价值证明（发票/合同）
   - 理赔申请表
4️⃣ **审核定损**：3个工作日内完成
5️⃣ **赔付到账**：审核通过后5个工作日内打款

【不予理赔的情况】
❌ 签收超过72小时
❌ 无法提供货值证明
❌ 包装不符合要求导致的损坏
❌ 不可抗力（地震、洪水等）

💡 建议：高价值货物务必保价，保价费很低但保障全面。
```

## 🎯 最佳实践

### 1. 报价技巧

✅ **推荐做法**：
- 先了解客户月运量，再报折扣价
- 同时提供零担和整车方案，让客户选择
- 明确报价有效期（旺季3天，淡季7天）
- 主动说明燃油附加费和保价费用

❌ **避免做法**：
- 不要直接报最低价，留谈判空间
- 不要忘记说明免责条款
- 不要承诺超出权限的折扣（8折以下需审批）

---

### 2. 客户跟进

✅ **推荐做法**：
- A级客户：每周跟进一次
- B级客户：每两周跟进一次
- C级客户：每月问候一次
- 记录每次沟通内容，建立客户档案

❌ **避免做法**：
- 不要频繁打扰客户（一周超过2次）
- 不要只谈业务，适当关心客户近况
- 不要忽略客户投诉，24小时内必须回应

---

### 3. 异议处理

✅ **推荐做法**：
- 先倾听，理解客户真实顾虑
- 用数据和案例说话，增强说服力
- 提供多种方案，让客户有选择权
- 适时让步，但不要一次性让到底

❌ **避免做法**：
- 不要与客户争辩
- 不要轻易承诺做不到的事
- 不要贬低竞争对手
- 不要忽视客户的隐性需求
