# 物流销售 Agent Skill - 完整实现方案

## 📋 项目概览

这是一个专为**内部物流行业 ToB 销售**设计的智能助手 Skill，帮助销售人员更高效地工作。

### 核心功能

1. **智能运价查询** - 支持零担(LTL)和整车(FTL)，自动计算折扣
2. **实时运单追踪** - 查询货物位置、状态、异常处理
3. **客户管理** - 客户信息查询、跟进提醒、档案管理
4. **销售话术** - 应对价格异议、竞品对比等专业话术
5. **业务FAQ** - 禁运品、理赔规则、保价政策等常见问题
6. **报价单生成** - 自动生成正式报价文档（含客户信息、服务承诺）

## 🎯 使用场景

### 场景 1: 快速报价

**用户输入**：
```
帮我查一下从上海到北京的运价，300kg
```

**Agent 工作流程**：
1. 识别意图 → 匹配 `logistics-sales-agent` Skill
2. 提取参数 → from=上海, to=北京, weight=300
3. 调用脚本 → `python scripts/quote.py --from 上海 --to 北京 --weight 300`
4. 解析结果 → 格式化输出零担/整车方案
5. 主动追问 → "需要保价吗？货值多少？"

**输出示例**：
```

─────────────────────
【零担 300kg】
  单价：12元/kg
  基础运费：3,600元
  燃油附加费：288元（8%）
  ✅ 合计：3,888元
  ⏱ 时效：3-4天

【整车方案】如货量更大可选整车：
  • 4.2米厢车：4,104元（含燃油），限重3吨
  • 9.6米厢车：7,020元（含燃油），限重10吨
─────────────────────
💡 月运费超10万可享9折优惠，欢迎询问框架合同

需要保价吗？保价费为货值的0.3%，最低50元。
```

## 🔧 技术实现

### 1. 脚本设计原则

#### quote.py（运价查询）
- **输入**：起点、终点、重量、类型、月运量
- **输出**：JSON 格式的报价数据或报价单文档
- **特性**：
  - 支持零担/整车两种模式
  - 自动计算燃油附加费（8%）
  - 根据月运量计算阶梯折扣
  - 可生成正式报价单文档

#### track.py（运单追踪）
- **输入**：运单号
- **输出**：JSON 格式的运单状态和轨迹
- **特性**：
  - 模拟真实运单数据（5个演示单号）
  - 未知单号生成随机在途状态
  - 标记异常状态并提供处置建议

#### customer.py（客户管理）
- **输入**：操作类型（list/get/followup/search）
- **输出**：JSON 格式的客户信息
- **特性**：
  - 5个模拟客户档案（不同等级）
  - 自动计算跟进提醒（基于下次跟进日期）
  - 支持关键词搜索（名称/联系人/城市/类型）

---

### 2. 数据模拟

所有数据均为模拟数据，便于演示和测试：

**运价数据** (`FREIGHT_RATES`)：
- 覆盖主要线路：上海↔北京/广州/成都/武汉
- 零担：按重量区间定价（0-100kg, 100-500kg, 500-1000kg, 1000-3000kg）
- 整车：按车型定价（4.2米/9.6米/17.5米）

**客户数据** (`CUSTOMERS`)：
- 5个客户，涵盖不同等级（A/B/C）和行业（电商/制造/零售/贸易/出版）
- 包含完整的联系信息、业务数据、跟进记录

**运单数据** (`MOCK_WAYBILLS`)：
- 4个演示运单：已签收/派送中/运输中/异常
- 每个运单包含完整轨迹（5-8个节点）

### 3. 与 DeepAgent 集成

Skill 通过以下方式与 DeepAgent 无缝集成：

1. **SKILL.md 元数据**：
   ```yaml
   name: logistics-sales-agent
   description: >
     物流公司内部销售智能助手...
   tags: [logistics, sales, B2B, quote, tracking]
   ```

2. **脚本调用约定**：
   - 所有脚本支持命令行参数
   - 输出统一为 JSON 格式
   - 错误信息也通过 JSON 返回

3. **渐进式披露**：
   - L1: SKILL.md frontmatter（始终加载）
   - L2: SKILL.md 全文（匹配后加载）
   - L3: references/ 目录（按需加载）
   - L4: scripts/ 目录（执行时调用）

## 📊 业务规则

### 1. 报价规则

| 规则 | 说明 |
|------|------|
| 燃油附加费 | 运价 × 8%，每月15日调整 |
| 报价有效期 | 标准7天，旺季（1/6-7/11月）3天 |
| 折扣权限 | 月均8折及以下需商务审批 |
| 零担上限 | 最大承运3吨，超出建议整车 |
| 保价费 | 货值 × 0.3%，最低50元 |

### 2. 理赔规则

| 情况 | 赔偿标准 |
|------|---------|
| 未保价 | 运费的3倍 |
| 已保价 | 货值的80% |
| 时效 | 签收72小时后不受理 |
| 免责 | 不可抗力（地震、洪水等） |

### 3. 禁运品

❌ **绝对禁运**（无例外）：
- 易燃易爆物品
- 武器弹药
- 走私品
- 毒品

⚠️ **限制运输**（需特殊手续）：
- 锂电池（需UN认证）
- 液体/粉末
- 贵重物品（建议保价）

## 🎓 最佳实践

### 1. 销售流程标准化

```
客户咨询 → 需求分析 → 方案报价 → 异议处理 → 成交签约 → 售后服务
   ↓           ↓           ↓           ↓           ↓           ↓
  FAQ解答   客户画像   智能报价   话术推荐   合同管理   跟进提醒
```

### 2. 客户分级管理

| 等级 | 月运费 | 跟进频率 | 折扣权限 | 服务级别 |
|------|--------|---------|---------|---------|
| A级 | >20万 | 每周 | 8-8.5折 | VIP专属客服 |
| B级 | 5-20万 | 每两周 | 8.5-9折 | 优先响应 |
| C级 | <5万 | 每月 | 9-9.5折 | 标准服务 |

### 3. 异议处理四步法

1. **倾听**：让客户充分表达顾虑
2. **共情**：表示理解客户的立场
3. **回应**：用数据和案例化解疑虑
4. **行动**：提供具体解决方案
